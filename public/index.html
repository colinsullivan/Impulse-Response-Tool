Reverberation Measurement Tool

This tool will help you measure the reverberant characteristics of a room and then apply these characteristics to any audio of your choosing, making it sound like it was recorded in the room.

How does it work?

First, we create an impulse response of the room using one of a few methods. The impulse response is an audio signal that will contain the reverberant characteristics of the room, which can then be applied to any arbitrary sound by process of convolution.

When can I do it?

Right now.

Get Started

Capturing Room Characteristics

There are a many different ways to capture the characteristics of the room. Please choose the method below that you would like to use.

  • Impulse (Balloon Pop)
  • Golay Codes
  • Allpass Chirps
  • Sine Sweeps

This is the simplest of the methods, you will record the response of the room to a loud sound of a balloon being popped, which is virtually an impulse.

Golay codes are a set of signals that relate to the impulse signal in a special way. The room will need to be measured against both Golay code components, which are then cross-correlated and summed together to estimate the impulse response of the room. This is an extra step, but ultimately yields a higher signal to noise ratio in the impulse response which is good.
